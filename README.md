## Features
- Seamless Integration - Displays seamlessly on user profiles with the native Steam styling
- Rank Badge - Shows your current rank with the official badges
- Quick Stats - Display quick stats such as KD and Win%
- Link to Tracker.GG page
- Caches results to disk, clear via the plugins settings page
- Works in Steam overlay

## Installation

1. Download the latest release from [Releases](https://github.com/Nazza1996/Millennium-R6-Tracker/releases/latest)
2. Extract to the Millennium plugins folder
3. Restart steam
4. Enable the plugin in Millenium settings

## Usage
1. Go to a user profile
2. Using the search box, like the profile to a Ubisoft id
3. Change the user or refresh with the available buttons
4. Stats will be automatically refreshed after 1 hour
5. You can clear the cache in the plugin's configuration menu
